# GAS Bill Calculator

Web application for gas consultants to compare a customer's previous gas bill
with a new offer and generate a printable PDF report.

## Features (MVP)
- Guided manual input of previous gas bill data
- Deterministic gas cost calculation
- Comparison between old and new bill
- Printable PDF for the customer

## No AI
This project does NOT use AI.
All calculations are deterministic and transparent.

## Target users
Gas and energy consultants.
